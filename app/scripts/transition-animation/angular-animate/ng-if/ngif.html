Explanation
  • class angular-animate-ngif-toggle has basic style and a linear transition property of 2s
  • class angular-animate-ngif-toggle.ng-enter has opacity 0, which means as you begin to start the transition, start at opacity 0
  • class angular-animate-ngif-toggle.ng-enter-active has opacity 1, which means as you finish the transition, make the final opacity = 1
  • class angular-animate-ngif-toggle.ng-leave has opacity 1, which means as you begin to leave the transition, start at opacity 1
  • class angular-animate-ngif-toggle.ng-leave-active has opacity 0, which means as you finish the transition, make the final opacity = 0
