Explanation
  • In the link function of the directive, find the element by class name to which you want to add animation.
  • Add a new class (eg: new-class-name) to that element using $animate.addClass
  • Angularjs adds new states to new-class-name, eg: new-class-name-add, new-class-name-add-active etc which can be used for animation
