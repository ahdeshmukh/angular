Isolate scope attr (@)
Here the fruit-name is considered as an attribute of the div, which is passed into the 'xyz' property of the isolate scope.
This 'xyz' property is then printed in the template which prints the string value of the 'xyz' property.
In the code we have var directive = {scope: {xyz: "@fruitName"}}
We get the value of fruit-name attribute from the div, assign it to the 'xyz' property in the isolate scope and then print it.
We can also do var directive = {scope: {fruitName: "@"}}, where instead of 'xyz' property name, we are using 'fruitName' which matches the attribute fruit-name of the div.
